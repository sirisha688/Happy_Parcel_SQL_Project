ğŸ§± Database Structure

The database consists of four tables:
1. Categories
Stores different product categories (Birthday Gifts, Wedding Gifts, etc.)

2. Products
Stores product details such as name, category, price, and stock quantity

3. Customers
Stores customer personal and contact information

4. Orders
Stores order details including order date, quantity, total amount, order status, and payment method

All tables are connected using Primary and Foreign Keys to maintain data integrity.
